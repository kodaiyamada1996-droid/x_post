転職活動、まだ「根性」でやってない？

僕は4社経験して、今はSaaS企業でインサイドセールスをやってるんだけど、

正直、最後の転職活動は**AIをフル活用して効率化したおかげ**で、圧倒的にラクだった。

今日はその全ステップを公開するね。

───

僕のキャリアを簡単に話すと、

新卒で大手住宅メーカーに入って、
通信系ベンチャーに転職して失敗して、
大手人材会社でボロボロになりながら成長して、
今のSaaS企業にたどり着いた。

4社、3回の転職。

正直、最初の転職は**完全に「逃げ」**だった。

住宅営業時代、展示場で所長に報告したら
「お前に聞いてねえよ」って言われたこともある。

それくらいメンタルを削られる環境で、
コロナ禍で時間ができた瞬間に逃げ出すように転職した。

結果、入社初日に「失敗した」と悟った。

教育担当の先輩はエクセルもまともに使えないし、
暇すぎて定時帰り。

年収も大幅ダウンで、
結婚とか将来のビジョンが全く見えなくなった。

───

この失敗から学んだのは、

**「転職は準備が9割」** ってこと。

で、3回目の転職（今の会社）では、やり方を完全に変えた。

何を変えたかというと、**AIを転職活動に組み込んだ**。

───

🔥 **根性転職 vs AI転職、こんなに違う**

**① 企業研究**

❌ 根性：IR資料を読み漁る → OpenWork見る → ニュース検索 → 3時間
✅ AI：Geminiに聞く → 3分

実際に使えるプロンプト👇

```
「[企業名]について以下を教えて。
・直近3年の売上推移と成長率
・主要プロダクトと競合との差別化ポイント
・社風や働き方の特徴
・業界内でのポジション
・最近のニュースやトピック」
```

これだけで、面接前に必要な情報の8割は揃う。
しかもGeminiはGoogle検索と連動してるから、最新情報も拾える。

───

**② 職務経歴書の作成**

❌ 根性：白紙から書く → 3日悩む → 結局ありきたりな表現
✅ AI：Geminiに壁打ち → 1時間で完成

プロンプト例👇

```
「以下の経歴をもとに、SaaS企業のインサイドセールス職に
応募する職務経歴書を作成してください。

特に以下のポイントを押さえてください：
- 実績は定量的に（数字で）表現する
- 前職の経験がSaaS営業にどう活きるか翻訳する
- NGワード（頑張りました、コミュ力）を避ける

【経歴】
（ここに自分の経歴を貼り付ける）」
```

僕の場合、住宅営業の経験を
「一生に一度の高額商材を扱う中で、
顧客の潜在ニーズを引き出す提案型営業を実践」
みたいにSaaS文脈に翻訳してもらった。

自分では思いつかなかった表現が出てくるのが、AIの強み。

───

**③ 面接対策**

❌ 根性：想定質問を考える → 一人で回答を暗記 → 本番でパニック
✅ AI：Geminiに面接官役をやらせる → 深掘りに慣れる

プロンプト例👇

```
「あなたはSaaS企業の面接官です。
インサイドセールス職の2次面接を行ってください。

候補者のプロフィール：
- 住宅営業→人材営業→SaaS志望
- 転職理由：ワークライフバランスと成長環境の両立

1問ずつ質問して、私の回答に対して深掘りしてください。
厳しめにお願いします。」
```

これ、マジで効果ある。

「なぜこの会社なのか？」の深掘りに対して、
スラスラ答えられるようになるまで壁打ちできる。

人材会社にいたからわかるけど、
面接の合否は**事前準備の質で8割決まる**。

そしてその事前準備を、AIで圧倒的に効率化できる。

───

**④ 志望動機のブラッシュアップ**

❌ 根性：「御社の成長性に惹かれ…」→ どの会社にも言える薄い内容
✅ AI：企業のプロダクト特性に合わせて深掘り → 「なぜこの会社か」が明確に

プロンプト例👇

```
「以下の企業情報をもとに、この会社ならではの
志望動機を3パターン作成してください。

判断軸として以下を使ってください：
① プロダクトがMUST型（経費精算等）かMORE型（SFA等）か
② 価格帯のスタンス（高単価 or 普及型）
③ MVV（ミッション・ビジョン・バリュー）
④ 私個人の転職軸との接点

【企業情報】
（Geminiで調べた情報を貼り付ける）

【私の転職軸】
・顧客貢献を実感できる環境
・データドリブンな営業組織
・ワークライフバランスとの両立」
```
